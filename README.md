#  Canvas Editor

A lightweight **Figma-like canvas editor** built using **HTML, CSS, and Vanilla JavaScript**.

This project lets you draw shapes and text, move, resize, rotate elements, manage layers, zoom and pan on an unlimited canvas, save designs locally, and export them as **JSON** or **HTML**.

---

##  Features

* Rectangle, circle, and text tools
* Select, move, resize, and rotate elements
* Properties panel

  * Fill color
  * Stroke color and width
  * Opacity
  * Font size (for text)
* Layer controls

  * Bring to front
  * Send to back
* Right-click context menu

  * Copy
  * Paste
  * Layer actions
* Zoom and pan with an infinite canvas feel
* Save design data to **Local Storage**
* Export designs as

  * **JSON** (layout data)
  * **HTML** (visual reproduction using inline styles)

---

## 🛠 Tech Stack

* **HTML**
* **CSS**
* **Vanilla JavaScript**

No external libraries or frameworks were used.

---

##  About This Project

This canvas editor was built **from scratch** to understand how real design tools work internally.

While building it, I focused on:
* Element rendering and re-rendering
* Layer management and z-index handling
* Data-driven state (saving and restoring layouts)
* Implementing editor-like interactions without libraries

The goal was learning and control, not shortcuts.
